**Input** — labelled dark text field; neon focus ring.

```jsx
<Input label="Как вас зовут" placeholder="Алексей" />
<Input label="Как связаться" hint="Telegram / почта / телефон" />
<Input label="E-mail" error hint="Проверь адрес" defaultValue="bad@" />
```

Mono uppercase label, inset dark field, accent border + glow on focus. Pass `error` for the error state.
