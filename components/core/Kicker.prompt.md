**Kicker** — the section eyebrow: two-digit accent index + a short mono label.

```jsx
<Kicker index="01">Почему сайты молчат</Kicker>
<Kicker index="04" rule>Работы</Kicker>
```

The defining header motif across lstmind. `index` shows in accent; `rule` adds a hairline between
index and label. Pair above every `SectionHeader`.
