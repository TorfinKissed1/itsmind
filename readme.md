# lstmind — сайт-портфолио

Тёмный одностраничный лендинг веб-разработчика (Алексей, «lstmind» — «сайты, которые работают»):
hero с 3D-объектом, плавный скролл, анимации появления текста, горизонтальная карусель
работ, секции услуг и процесса, форма заявки.

**Живая версия:** https://torfinkissed1.github.io/itsmind/

## Запуск локально

Статический сайт без сборки: React / GSAP / Lenis / three.js подключены с CDN, JSX
компилируется прямо в браузере — поэтому нужен HTTP-сервер (открыть файл напрямую через
`file://` не получится).

Из корня проекта:

```bash
python _devserver.py
```

затем открыть http://127.0.0.1:8000/ui_kits/website/index.html

## Структура

- `ui_kits/website/` — сам сайт: `index.html`, стили `kit.css`, контент `data.js`,
  секции `parts1–3.jsx`, эффекты `fx.jsx` / `fx2.jsx`, иконки `svgs.jsx`.
- `_ds_bundle.js`, `styles.css`, `tokens/` — UI-компоненты, токены и базовые стили.
- `assets/`, `uploads/` — изображения (текстуры, дым, чернила).
